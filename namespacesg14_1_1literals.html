Detailed Description

generate an sg14::elastic object using a literal

Returns
the given value represented using an sg14::elastic type
Note
The return type is guaranteed to be no larger than is necessary to represent the value.
Example

To define an int-sized object with value 1536:

using namespace sg14::literals;
constexpr auto n = 1536_elastic;
static_assert(n==1536, "n now has the value, 1536");
static_assert(std::is_same<decltype(n), const elastic<11, -9>>::value, "type only uses 1 bit of range");
